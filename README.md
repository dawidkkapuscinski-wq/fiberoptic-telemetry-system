# FiberOptic Telemetry System

Fiber optic and LTE-based communication system for video transmission, MAVLink telemetry and remote monitoring in UAV/UGV platforms.

## Project overview

This project was designed as a modular communication system for unmanned and mobile platforms.  
It combines video transmission, telemetry routing, LTE connectivity, ZeroTier overlay VPN and fiber optic / Ethernet communication.

## Main technologies

- Raspberry Pi
- Linux
- Fiber optic media converters
- Ethernet
- LTE modems
- ZeroTier
- MAVLink Router
- ffmpeg
- rpicam-vid
- UDP multicast
- SRT
- Bash automation

## Features

- Low-latency video transmission
- MAVLink telemetry routing
- Fiber optic / Ethernet communication
- LTE and ZeroTier integration
- Remote access to onboard devices
- Support for multiple receivers
- Automatic reconnect after network loss
- Linux autostart and watchdog logic
